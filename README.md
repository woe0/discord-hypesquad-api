#  Discord HypeSquad Tool

## ⚙️ طريقة الاستخدام

1. افتح ديسكورد من المتصفح.
2. اضغط على `F12` لفتح أدوات المطور (Developer Tools).
3. اذهب إلى تبويب **Console**.
> 💡 **ملاحظة:** في بعض المتصفحات، قد يتم منع اللصق كإجراء أمني. للسماح باللصق، اكتب `allow pasting` واضغط **Enter** أولاً.
4. انسخ الكود من ملفات المستودع، الصقه في الكونسول ثم اضغط **Enter**.

## 📝 ملاحظات التعديل

* **التوكن:** قم باستبدال عبارة `YOUR_TOKEN_HERE` بالتوكن الخاص بحسابك.
* **رقم الشارة:** الرقم `1` الموجود في الكود يمثل نوع الشارة، يمكنك تغييره حسب الجدول التالي:

| الرقم | اسم الشارة | اللون |
| :---: | :--- | :--- |
| **1** | House of Bravery | 🟣 البنفسجية |
| **2** | House of Brilliance| 🔴 الحمراء |
| **3** | House of Balance | 🟢 الخضراء |

> ⚠️ **تحذير أمني خطير:** لا تشارك التوكن الخاص بك مع أي شخص نهائياً، لأنه يعطي وصولاً كاملاً لحسابك.

# 🛡️ Discord HypeSquad Tool (English)

## ⚙️ How to Use

1. Open Discord in your browser.
2. Press `F12` to open Developer Tools.
3. Go to the **Console** tab.
> 💡 **Note:** In some browsers, pasting might be blocked as a security measure. To allow pasting, type `allow pasting` and press **Enter** first.
4. Copy the code from the repository files, paste it into the console, and press **Enter**.

## 📝 Modification Notes

* **Token:** Replace `YOUR_TOKEN_HERE` with your account's token.
* **Badge Number:** The number `1` in the code represents the badge type. You can change it according to the following table:

| Number | Badge Name | Color |
| :---: | :--- | :--- |
| **1** | House of Bravery | 🟣 Purple |
| **2** | House of Brilliance| 🔴 Red |
| **3** | House of Balance | 🟢 Green |

> ⚠️ **Security Warning:** Never share your token with anyone, as it grants full access to your account.
